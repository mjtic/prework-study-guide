HTML

The following element types are commonly used to add content to pages:
  1. Text elements
  2. Image elements
  3. Header elements
  4. List elements
  • The head element contains information about the webpage, or metadata, that helps the browser to render the page correctly.
  • The body element represents the visible content shown to the user.

    GIT

    • git status: check what branch we are currently on
    • git checkout -b branch-name: creates a new branch and switches to it (ex. git checkout -b feature/add-html, git checkout feature/add-html)
    • git add -A: command to commit all files in the working branch
    • git commit -m "": we use the -m flag to associate a message with our commit (short and descriptive messages
    • git pull origin main: use following command in your terminal to pull in the base branch (note: it is a good habit to always pull the base branch into your feature branch before pushing up any code to main branch)
    • git push origin feature/add-html: git pushes changes to feature/add-html branch
